## About APP

<h3>Create CRUD API</h3>

#### Clone Application
#### Run composer install
#### Create .env
#### Create Database
#### Run Migration
##### Get all books: /api/books :GET
##### Create books: /api/books :POST
##### Read a book: /api/books/id :GET
##### Update a book: /api/books/id :PATCH
##### Delete a book: /api/books/id :DELETE
